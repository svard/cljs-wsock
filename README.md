# cljs-wsock

Clojurescript wrapper for goog.net.WebSocket.

A small project for teaching myself clojurescript and core.async library.

## Usage

```sh
lein install
```
and add
```clojure
[cljs-wsock "0.1.0-SNAPSHOT"]
```
as dependency in project.clj
